### Compile o código
Abra um terminal, navegue até o diretório src e compile todos os arquivos Java:
```bash
cd EngenhariaSoftware3
```
```bash
cd Facade
```
```bash
cd src
```
```bash
javac *.java
```
Isso criará arquivos .class para cada uma das classes.

### Execute o programa
```bash
java Main
```
